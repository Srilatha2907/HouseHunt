# HouseHunt

HouseHunt is a modern house rental and property listing platform built using the MERN stack. The application simplifies the process of finding, renting, and managing properties by providing an intuitive and user-friendly experience.

## Features

* User Authentication and Authorization
* Property Listing Management
* Advanced Property Search and Filtering
* Property Details with Images
* Responsive User Interface
* Secure Backend APIs
* Real-Time Property Updates
* User Dashboard for Managing Listings

## Tech Stack

### Frontend

* React.js
* HTML5
* CSS3
* JavaScript

### Backend

* Node.js
* Express.js

### Database

* MongoDB

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/srilatha2907/househunt.git
   ```

2. Navigate to the project directory:

   ```bash
   cd househunt
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Configure environment variables in a `.env` file.

5. Start the application:

   ```bash
   npm start
   ```
